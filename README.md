# mongo-scraper
Click [here](https://arcane-dusk-24699.herokuapp.com/) for a live version of this app!

# Description 
This app allows users to scrape articles from the Memeburn website and create notes for each article.

* MongoDB
* Mongoose
* Handlebars.js
* Javascript/ES6
* jQuery
* AJAX/JSON
## NPM modules:
